## **📌 Project Description**
This project is a **text mining tool** for analyzing scientific abstracts from **PubMed** to identify mentions of **diseases, microbes, or bacteria** in the context of **aquaculture**. It allows users to:
- Search for **specific diseases or microbes** in PubMed abstracts.
- Count **mentions of microbes** in relation to **fish species** (e.g., tilapia, salmon, shrimp).
- Visualize the results using **bar charts** with distinct colors for each fish species.
- Generate **regex patterns** automatically from user input to capture variations of scientific names (e.g., "Enterocytozoon hepatopenaei" and "E. hepatopenaei").

## **✨ Features**
✅ **PubMed API Integration** – Fetches abstracts from PubMed using the E-utilities API.
✅ **Regex Pattern Generation** – Automatically generates regex patterns from user input to match variations of scientific names.
✅ **Text Mining** – Counts mentions of microbes and their association with fish species.
✅ **Data Visualization** – Creates bar charts to visualize the frequency of mentions.
✅ **Customizable Input** – Users can input any disease or microbe of interest.
✅ **CSV Export** – Results can be saved to a CSV file for further analysis.
